# Adobe Concurrency Monitoring Services 3.6.0 Release Notes {#cm-services-rns}

This page describes new features, changes, and known issues with this release.

## Release Info {#rel-info}

Version: 3.6.0
Date: 02/04/2025

## Release Overview {#release-overview}

* Update API to include details from streams running on other tenant's apps.
* Add termination code to API method.
* Update metadata API by adding metadata for 409 response and modifiable metadata attribute.
