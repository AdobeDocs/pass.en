# [!UICONTROL Premium Workflow] overview

>[!IMPORTANT]
>
> Make sure you stay informed about the latest Adobe Pass product announcements and decommissioning timelines aggregated in the [Product Announcements](/help/authentication/product-announcements.md) page.

[!UICONTROL Premium Workflow] from Adobe Pass offers a robust suite of tools to improve authentication and streaming reliability. Gain real-time visibility with [!UICONTROL Entitlement Service Monitoring], ensure uptime with the smart Degradation API, and provide flexible access via [!UICONTROL TempPass]. 

Enhance the user experience with [!UICONTROL Home-Based Authentication] and Single Sign-On, while the MVPD Status Service proactively tracks and alerts on endpoint performance. These solutions empower programmers and MVPDs with actionable insights, optimize operations, boost engagement, and ensure uninterrupted content delivery&mdash;even during peak traffic.
